# Student Grade Calculator

A Java console application that calculates student grades across 5 subjects.

## Features
- Accepts marks for Mathematics, English, Computer Science, Physics and History
- Validates input and rejects values outside 0–100
- Assigns letter grades (A–F) and remarks
- Prints a formatted report card
- Supports multiple students in one session

## How to run
javac GradeCalculator.java
java GradeCalculator
